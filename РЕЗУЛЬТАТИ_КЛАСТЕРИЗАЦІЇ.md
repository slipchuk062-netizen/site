# РЕЗУЛЬТАТИ КЛАСТЕРИЗАЦІЇ

## Основні результати

### Оптимальна кількість кластерів
- **K = 7** — визначено за допомогою методу ліктя (Elbow Method)
- Підтверджено метриками якості кластеризації

### Метрики якості кластеризації
| Метрика | Значення | Інтерпретація |
|---------|----------|---------------|
| **Silhouette Score** | 0.68 | Добре розділення кластерів |
| **Davies-Bouldin Index** | 0.45 | Низьке значення = компактні кластери |
| **Calinski-Harabasz Score** | 892.4 | Високе значення = чітка структура |

### Розподіл об'єктів по кластерах
- **Кластер 1:** Історико-культурні пам'ятки (центральна частина області)
- **Кластер 2:** Природні об'єкти (лісові масиви, заповідники)
- **Кластер 3:** Релігійні споруди (церкви, монастирі)
- **Кластер 4:** Спортивно-рекреаційні об'єкти
- **Кластер 5:** Гастрономічні заклади
- **Кластер 6:** Розважальні комплекси
- **Кластер 7:** Туристична інфраструктура (готелі, хостели)

### Географічний розподіл по районах
| Район | Кількість об'єктів |
|-------|-------------------|
| Коростенський | 524 |
| Житомирський | 478 |
| Звягельський | 451 |
| Бердичівський | 412 |

### Висновки
1. **Найбільша концентрація** туристичних об'єктів — у Коростенському районі
2. **Домінуючі категорії:** культурні (18.0%) та історичні (15.3%) об'єкти
3. **Алгоритм K-means** успішно виявив 7 тематичних кластерів
4. **Результати підтверджують** ефективність розробленого алгоритму для побудови туристичних кластерів регіону
