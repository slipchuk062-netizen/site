#### 1.2. Методика визначення оптимального K
**ЩО НОВО:**
Розроблено методику багатокритеріальної оцінки для визначення оптимальної кількості кластерів туристичних об'єктів:

**Формула комплексної оцінки:**
```
Quality_Score = (α × Silhouette) + (β × (1 - Davies-Bouldin_normalized)) + (γ × Calinski-Harabasz_normalized)

де:
α, β, γ - вагові коефіцієнти (α + β + γ = 1)
α = 0.4 (найважливіша метрика - якість кластерів)
β = 0.3 (сепарація між кластерами)
γ = 0.3 (щільність всередині кластерів)
```

**ТРАДИЦІЙНИЙ ПІДХІД:**
- Використання тільки Elbow Method
- Суб'єктивне визначення "ліктя"
- Одна метрика (WCSS)

**НАШ ПІДХІД:**
- Три незалежні метрики
- Об'єктивна оцінка якості
- Візуальне підтвердження через графіки

**РЕЗУЛЬТАТ:**
- K=7 з Quality_Score = 0.753
- Silhouette: 0.693 (відмінно)
- Davies-Bouldin: 0.620 (добре)
- Calinski-Harabasz: 1247 (високо)
